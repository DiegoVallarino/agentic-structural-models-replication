# Agentic Structural Models Replication Package

This repository contains the replication materials for an agent-based economic model with endogenous learning, network interactions, and counterfactual policy experiments.

## Contents

- `code/`: main simulation code
- `results/`: generated outputs
- `figures/`: final figures
- `docs/`: replication notes

## Replication goal

The objective of this repository is to allow an external reader to reproduce the main simulation results, summary tables, and figures.

## Current status

Initial repository structure created. Documentation and execution scripts are being organized.
