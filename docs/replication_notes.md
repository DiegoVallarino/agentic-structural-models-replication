# Replication notes

This repository is organized to allow external verification of the paper's main results.

## Main files

- `paper/manuscript.pdf`: main paper
- `paper/online_appendix.pdf`: technical appendix
- `code/main.py`: main simulation script

## Intended outputs

The replication package will produce:
- episode-level outputs
- summary tables
- final figures
- counterfactual comparisons

## Important note

The repository will distinguish between:
1. a paper-exact baseline specification
2. an extended laboratory version of the model
